# project22 — 경제 자립 단계별 AI 코칭

AI Reboot Academy 팀 프로젝트 (PROJECT 22) · Vite + React + TypeScript + Supabase(선택)

- 배포: https://aebonlee.github.io/project22/
- 구현 예시: 주제별 인터랙티브 데모 + 프로젝트 정보(문제·기능·기술) 포트폴리오

```bash
npm install && npm run dev      # 로컬 실행
npm run build                   # 빌드(dist)
```

OpenAI 키는 앱 내에서 입력(브라우저 저장)하거나 `VITE_OPENAI_API_KEY` 환경변수로 주입합니다. 키 없이도 모든 기능이 오프라인 샘플로 동작합니다.
